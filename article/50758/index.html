添加机器人好友

从群成员列表添加机器人可莉为好友
问题答案：逃跑的太阳

如遇到机器人没有同意添加好友的情况，估计是申请被过滤了
需要在群内发送指令#申请添加好友进行双向添加即可成功

绑定cookie(ck)和stoken(sk)

免责申明
如果账号被盗，被毁号之类的均与本人无关，本人没有那么大的能耐
一般情况下，即使别人拿到了你的cookie和stoken也不能做什么，毕竟应该没人会随便打验证码告诉别人吧？
插一嘴，不会真有人被盗号毁号了吧？
| 指令 | 说明 | | ----------- | ------------------------------------------------ | | #cookie帮助 | 查看获取ck的方法，将获取的的ck私聊发送给机器人 | | #扫码登录 | 该指令会直接获取sk，并且自动进行绑定 | | #刷新ck | 绑定了sk，如遇到ck失效的问题可以通过该指令刷新ck |
